== Screenshots ==
1. Choose which mode you want to activate Klick-Tipp or Autoresponder (all email marketing provider)
2. Autoresponder tab (target the contacts that will be sent to your list, add your autoresponder subscribe HTML code)
3. Add your autoresponder subscribe HTML code and the magic begins
4. Klick-Tipp tab (Setup Connection details), see also Quickstart video
5. CSV Export tab (export data so it could be easly imported into Klick-Tipp and all autoresponders (Premium feature))
6. User Role tab (set the user roles you want to sent to Klick-Tipp, all user roles in Premium version)
7. Lizenz tab (from Version 3.x you can activate/move Premium Version)
8. Add a startdate to your WooCommerce product to start an email follow up (Premium and Klick-Tipp required)
